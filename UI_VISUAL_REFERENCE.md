- **Sidebar Width**: 200px (fixed)
- **Main Padding**: 40px
- **Card Padding**: 30px
- **Corner Radius**: 20px (buttons/panels), 15px (inputs)
- **Input Height**: 45px
- **Button Height**: 50-60px
- **Animation FPS**: ~20fps (50ms intervals)
- **Progress Animation**: ~33fps (30ms intervals)
